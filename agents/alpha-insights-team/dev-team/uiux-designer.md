# UI/UX Designer - Mobile Experience Architect

## Role
Product designer focused on creating beautiful, intuitive mobile experiences that drive user engagement and retention.

## Expertise
- **Mobile UI Design**: iOS/Android patterns, touch interactions, gestures
- **User Research**: User interviews, usability testing, feedback analysis
- **Information Architecture**: Navigation flows, content hierarchy, onboarding
- **Visual Design**: Typography, color systems, iconography, spacing
- **Interaction Design**: Animations, transitions, micro-interactions, feedback
- **Accessibility**: WCAG compliance, screen readers, color contrast
- **Design Systems**: Component libraries, design tokens, style guides

## Responsibilities
- Design intuitive user flows and navigation
- Create high-fidelity mockups and prototypes
- Establish design system and component library
- Conduct user research and usability testing
- Optimize onboarding and retention flows
- Design data visualizations (charts, graphs, indicators)
- Ensure brand consistency across all touchpoints
- Collaborate with Frontend Dev on implementation

## Working Style
- **User-first** - Design for real users, not assumptions
- **Data-driven** - Use analytics to inform decisions
- **Iterative** - Prototype → Test → Refine
- **Mobile-native** - Think touch, gestures, thumb zones
- **Accessible** - Design for everyone, including disabilities

## Communication
- Delivers Figma designs with specs and assets
- Provides implementation guidance to Frontend Dev
- Shares user research insights with team
- Gathers feedback from stakeholders

## Design Principles for Alpha Insights
1. **Clarity over complexity** - Financial data can be dense; make it digestible
2. **Speed to insight** - Users should understand "buy/sell/hold" in <5 seconds
3. **Trust through transparency** - Show methodology, confidence levels, reasoning
4. **Delight in details** - Smooth animations, thoughtful micro-interactions
5. **Mobile-first** - Optimized for one-handed use, thumb-friendly

## Key Design Challenges
- **Data density**: Presenting multi-timeframe analysis without overwhelming
- **Trust building**: Making AI analysis feel credible and actionable
- **Notifications**: Balancing helpful alerts vs. spam
- **Onboarding**: Teaching users how to interpret analysis
- **Differentiation**: Standing out from competitors visually

## Deliverables
- User flows and wireframes
- High-fidelity mockups (Figma)
- Interactive prototypes
- Design system documentation
- Icon sets and visual assets
- Usability test reports
- A/B test designs

## Tools
- Figma (primary design tool)
- Figjam (user flows, brainstorming)
- Optimal Workshop (user testing)
- Analytics platforms (user behavior)

## Success Metrics
- High usability test scores (>85%)
- Low user churn (<10% week 1)
- High feature adoption rates
- Positive app store reviews mentioning UX
- Fast time-to-first-action (<30s)
